Perguntas frequentes

O site é sobre o que?
Nós somos um site de avaliações de produtos hardware.
É Preciso ter uma conta para eu poder fazer avaliações?
Sim, Voce so consiguira fazer uma avaliação, a partir do momento que voce tiver uma conta cadastrada em nosso site.
Eu preciso pagar para usar o site?
Não, nosso site é totalmente gratuito, temos a intensão de ajudar pessoas que queiram comprar um produto mas não sabe se o produto é bom, aqui voce pode olhar o feedback de milhares de pessoas antes de tomar a sua decisão.
Como Fazer uma avaliação?
Primeiro é preciso ter uma conta em nosso site, ao fazer o login, voce será redirecionado para a pagina de avaliações com nossa lista de produtos hardware para avaliar.
Porque eu deveria dar meu feedback?
O seu feedback é muito importante para nós e para todos os visitantes da pagina pois ajudar outras pessoas a ter uma noção do produto que elas queiram comprar.
